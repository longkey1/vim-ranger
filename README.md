# vim-ranger

## Configuration

* Set ranger executable path: `let g:ranger_executable = 'ranger'`
* Custom command to run when ranger exits: `let g:ranger_on_exit = ''`
* Command used to open files: `let g:ranger_open_mode = 'tabe'`

## Requirement

ranger >= 1.5.1
